# PromptCraft 技术规范文档

## 1. 技术架构设计

### 1.1 整体架构
```
用户界面 (HTML/CSS/JS)
    ↓
API层 (Express.js)
    ↓
业务逻辑层 (JavaScript)
    ↓
数据层 (JSON文件)
```

### 1.2 技术选型原则
- **简单优先**：选择学习成本低、开发速度快的技术
- **稳定可靠**：使用成熟的技术栈，避免前沿但不稳定的工具
- **轻量化**：最小化依赖，减少打包体积
- **快速迭代**：支持快速开发和部署

## 2. 前端技术规范

### 2.1 技术栈
- **HTML5**：语义化标签
- **CSS3 + Tailwind CSS + DaisyUI**：通过CDN引入，利用DaisyUI组件库加速开发
- **JavaScript + Alpine.js (v3.x)**：使用轻量级框架处理状态管理和DOM更新，替代原生JS
- **Fetch API**：原生HTTP请求
- **打包工具**：`pkg` (用于生成 Windows .exe 可执行文件)
- **辅助工具**：`open` (用于服务启动后自动打开浏览器)


### 4.3 错误处理规范
```javascript
// 前端错误处理
try {
  const result = await generatePrompts(input);
  displayResults(result.data);
} catch (error) {
  showError('生成失败，请重试');
  console.error('Generation failed:', error);
}

// 后端错误处理
const generatePrompts = async (req, res, next) => {
  try {
    const { input } = req.body;
    
    if (!input || input.trim().length === 0) {
      return res.status(400).json({
        success: false,
        error: {
          code: 'INVALID_INPUT',
          message: '输入内容不能为空'
        }
      });
    }
    
    const results = await promptGenerator.generate(input);
    res.json({
      success: true,
      data: results
    });
    
  } catch (error) {
    next(error);
  }
};
```

## 5. 性能优化规范

### 5.1 前端优化
```javascript
// 防抖处理用户输入
const debounce = (func, wait) => {
  let timeout;
  return function executedFunction(...args) {
    const later = () => {
      clearTimeout(timeout);
      func(...args);
    };
    clearTimeout(timeout);
    timeout = setTimeout(later, wait);
  };
};

// 缓存DOM查询
const elements = {
  input: document.getElementById('prompt-input'),
  generateBtn: document.getElementById('generate-btn'),
  resultsContainer: document.getElementById('results-container')

// 延迟加载非关键资源
```

### 5.2 后端优化
```javascript
// 响应压缩
const compression = require('compression');
app.use(compression());

// 静态资源缓存
app.use(express.static('public', {
  maxAge: '1d'

// API缓存（内存缓存）
const cache = new Map();
const getCachedResult = (key) => cache.get(key);
const setCachedResult = (key, value) => {
  cache.set(key, value);
  // 限制缓存大小

    const firstKey = cache.keys().next().value;

  }
};
```
