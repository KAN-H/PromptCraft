# PromptAtelier 提示词生成器 - 6小时开发版 PRD

## 1. 产品概述

### 1.1 产品名称
PromptAtelier 提示词生成器（精简版）

### 1.2 产品定位
轻量化、高效的AI提示词生成工具，专注核心功能，6小时内完成开发部署。
**新增特性**：采用"结构化提示词"理念，兼容开源社区标准，支持无构建流程的现代化开发。

### 1.3 目标用户
- AI开发者
- 内容创作者
- 营销人员

## 2. 核心功能（MVP版本）

### 2.1 用户输入模块
- 单一文本输入框（最大500字符）
- 输入验证和字符计数
- 一键清空功能

### 2.2 提示词生成模块
- 生成3种风格的提示词（而非5种）：
  1. **专业逻辑型**：严谨、符合逻辑
  2. **创意激发型**：富有想象力、开放性
  3. **简洁实用型**：直接、高效
- **结构化组装**：基于 [角色] + [任务] + [约束] 的结构生成
- 3秒内响应
- 加载状态显示

### 2.3 提示词展示模块
- 3个结果卡片展示
- 每个卡片显示风格标签
- **结构化视图**：高亮显示提示词的关键部分
- 单个复制按钮
- **JSON格式复制**：支持一键复制为API可用的JSON格式
- 全部复制按钮
- 重新生成按钮

## 3. 技术架构（简化版）

### 3.1 前端技术栈
- **HTML5 + Alpine.js**（引入轻量级响应式框架，替代繁琐的原生DOM操作）
- **Tailwind CSS + DaisyUI**（使用组件库加速UI开发，CDN引入）
- **Fetch API**（原生HTTP请求）
- **单页应用**（无路由需求，无构建步骤）

### 3.2 后端技术栈
- **Node.js + Express**
- **内存数据存储**（去除MongoDB依赖）
- **预定义模板库**（JSON文件存储，兼容 `awesome-chatgpt-prompts` 格式）
- **RESTful API**

### 3.3 部署方案
- **本地桌面版封装**（针对低配设备优化）
  - 使用 `pkg` 工具将 Node.js 后端与前端资源打包为单文件 `.exe`。
  - **运行机制**：双击 exe -> 启动轻量级本地服务 -> 自动调用系统浏览器打开界面。
  - **资源占用**：极低（仅占用 Node.js 运行时内存，约 50MB），适合 i5-7200U/8GB 配置。
  - **零依赖**：无需安装 Node.js 或 Docker。

## 4. 业务流程

### 4.1 用户使用流程
1. 用户下载并解压缩 `PromptAtelier.zip`。
2. 用户双击 `PromptAtelier.exe` 启动程序。
3. 程序在后台自动启动本地服务。
4. 默认浏览器自动打开 `http://localhost:3000`。
5. 用户在网页上输入内容，选择提示词风格。
6. 系统生成提示词并展示。
7. 用户可复制提示词或重新生成。

### 4.2 错误处理流程
- **输入错误**：
  - 系统提示“输入内容超出限制（500字符）”，并清空输入框。
- **生成失败**：
  - 提示“提示词生成失败，请重试。”
  - 后台记录错误日志，便于开发者排查。
