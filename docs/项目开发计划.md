#### 7.1 什么是 Agent Skills？

Agent Skills 是 Anthropic 推出的**开放标准**，用于扩展 AI Agent 的专业能力。

**核心概念**：
```
Skills = SKILL.md（指令） + references/（文档） + assets/（脚本）
```

**关键特性**：
- **渐进式加载**：元数据先加载，完整指令按需加载
- **跨平台兼容**：同一 Skill 可在 Claude.ai、Claude Code、API 中使用
- **自动触发**：根据任务语义自动激活相关 Skill
- **社区生态**：与 agentskills.io 生态互通

#### 7.2 集成价值

| 价值 | 说明 |
|------|------|
| 能力扩展 | 用户可添加自定义专业 Skills |
| 知识复用 | 设计知识可打包、分享、复用 |
| 社区生态 | 支持导入社区 Skills |
| 专业化 | 每个设计领域有专门的 Skill |

> **来源**：参考 `docs/项目借鉴分析报告.md` 中的提示词类型分类体系

#### 5.1 Skills 功能概述
**目标**：引入技能型（SKILL）提示词分类，使 AI 具备特定专业技能

**提示词类型体系**：
```
提示词类型 (type):
├── TEXT        # 文本输出
├── IMAGE       # 图像生成 ✅ 当前核心
├── VIDEO       # 视频生成
├── AUDIO       # 音频生成
├── SKILL       # 技能型提示词 🆕 待开发
└── STRUCTURED  # 结构化(JSON/YAML)
```

#### 5.2 Skills 功能设计

**什么是技能型提示词？**
技能型提示词不直接生成内容，而是赋予 AI 特定的专业技能，让它能够持续以该角色/技能进行交互。

**设计领域 Skills 示例**：

| 技能名称 | 描述 | 应用场景 |
|---------|------|---------|
| `logo-critique-expert` | Logo设计评审专家 | 分析现有Logo的优缺点，提供改进建议 |
| `brand-strategy-advisor` | 品牌战略顾问 | 从品牌定位角度指导设计方向 |
| `color-theory-master` | 色彩理论大师 | 专业配色建议、色彩心理学分析 |
| `typography-specialist` | 字体设计专家 | 字体选择、排版建议 |
| `design-trend-analyst` | 设计趋势分析师 | 分析当前设计趋势，预测未来方向 |
| `visual-hierarchy-expert` | 视觉层次专家 | 指导信息架构和视觉权重分配 |
| `accessibility-consultant` | 无障碍设计顾问 | 确保设计符合无障碍标准 |
| `print-production-expert` | 印刷工艺专家 | 印刷材质、工艺选择建议 |

#### 5.3 Skills 技术实现

**数据结构设计**：
```json
{
  "skillId": "logo-critique-expert",
  "name": "Logo设计评审专家",
  "category": "design-skill",
  "description": "专业分析Logo设计的优缺点",
  "systemPrompt": "你是一位资深的Logo设计评审专家...",
  "capabilities": [
    "分析Logo的视觉平衡",
    "评估品牌识别度",
    "提供改进建议"
  ],
  "inputFormat": "上传Logo图片或描述现有设计",
  "outputFormat": "结构化评审报告"
}
```

**实现步骤**：
1. **Phase 5.3.1** - 数据结构定义
   - 创建 `data/skills.json` 存储技能定义
   - 创建 `schemas/skill_schema.json` 定义数据规范

2. **Phase 5.3.2** - 后端服务
   - 创建 `backend/services/skillService.js`
   - 添加 Skills 相关 API 端点
   - 实现技能激活和切换逻辑

3. **Phase 5.3.3** - 前端集成
   - 在设计助手中添加"专家技能"选择器
   - 技能组合使用（如同时激活多个技能）

4. **Phase 5.3.4** - 技能与模板联动
   - 预设模板可以关联推荐技能
   - 根据用户输入自动推荐相关技能

**涉及文件**：
- `data/skills.json` 🆕 - 技能定义数据
- `schemas/skill_schema.json` 🆕 - 技能数据规范
- `backend/services/skillService.js` 🆕 - 技能服务
- `backend/routes/prompts.js` - 添加 Skills API
- `frontend/index.html` - 技能选择器 UI

用户选择收藏 → 注入系统提示词 → AI 学习风格 → 生成相似质量的提示词
