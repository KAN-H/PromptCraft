# 🔥 智能动态系统提示词功能说明

## 功能概述

**智能动态系统提示词**是一项创新功能，能够根据用户输入的关键词自动调整AI的系统提示词，从而生成更加专业、精准、符合用户需求的图像生成提示词。

## 工作原理

### 1. 用户输入分析
系统会智能分析用户输入的所有内容，包括：
- 模板变量（如品牌名称、设计风格等）
- 附加输入（用户补充的关键词）
- 模板基础内容

### 2. 关键词识别

系统内置了三大识别规则库：

#### **设计风格识别**
- **极简风格 (Minimalist)**: 检测"简约"、"极简"、"简洁"等关键词
  - 自动调整：注重负空间、简洁几何、有限色彩、功能与美感平衡

- **现代风格 (Modern)**: 检测"现代"、"当代"、"时尚"等
  - 自动调整：大胆色彩、流畅线条、视觉冲击、时代感

- **复古风格 (Vintage)**: 检测"复古"、"怀旧"、"经典"等
  - 自动调整：怀旧色调、经典设计、历史感

- **奢华风格 (Luxury)**: 检测"奢华"、"高端"、"精致"等
  - 自动调整：金属质感、优雅字体、黑白金配色

- **活泼风格 (Playful)**: 检测"活泼"、"有趣"、"俏皮"等
  - 自动调整：明快色彩、活泼造型、卡通元素

- **专业风格 (Professional)**: 检测"专业"、"商务"、"正式"等
  - 自动调整：严谨设计、稳重色彩、强调可信度

- **有机风格 (Organic)**: 检测"自然"、"有机"、"温暖"等
  - 自动调整：自然曲线、大地色系、亲和力

- **科技风格 (Tech)**: 检测"科技"、"数字"、"未来"等
  - 自动调整：几何化、科技蓝、渐变效果、未来感

#### **行业特性识别**
- **餐饮行业 (Food)**: 暖色调、食欲感、食材元素
- **时尚行业 (Fashion)**: 时尚感、细节刻画、高级感
- **科技行业 (Tech)**: 现代视觉、几何图形、冷色调
- **美容行业 (Beauty)**: 柔和优雅、粉金色系、质感表现
- **教育行业 (Education)**: 明快色彩、知识元素、信任感
- **健康行业 (Health)**: 清新干净、绿蓝色系、活力感

#### **视觉元素识别**
- 火焰元素、几何图形、文字排版、手绘插画、渐变效果等

### 3. 动态提示词构建

系统根据识别结果，动态构建包含以下部分的系统提示词：

```
1. 基础角色定位（AI图像生成专家）
2. 检测到的设计风格倾向及详细指导
3. 检测到的行业特性及专业建议
4. 建议融入的视觉元素
5. 详细的输出要求和格式规范
6. 参考示例
```

## 使用示例

### 示例 1：烧烤店Logo设计

**用户输入：**
```
品牌名称：老李烧烤
设计风格：简约、现代
核心元素：烟火气、火焰
附加说明：体现烟火气和街头美食氛围
```

**系统识别：**
- ✅ 风格：Minimalist（简约）、Modern（现代）
- ✅ 行业：Food（餐饮）
- ✅ 元素：Flame（火焰）

**动态系统提示词片段：**
```
【设计风格倾向】
- MINIMALIST: 采用极简主义设计理念，注重负空间运用，
  使用简洁的几何形状和有限的色彩，强调功能性与美感的平衡
- MODERN: 运用当代设计趋势，大胆的色彩搭配，流畅的线条，
  注重视觉冲击力和时代感

【行业特性考量】
- FOOD: 注重食欲感表达，使用暖色调（红、橙、黄），
  可融入食材或烹饪元素，传达美味和品质

【建议融入的视觉元素】
- flame, geometric, typography
```

**AI最终输出：**
```
Minimalist modern BBQ restaurant logo design, 
stylized flame icon with geometric grill elements, 
warm orange-red gradient color palette, 
bold sans-serif typography "老李烧烤", 
clean vector graphic style with simple lines, 
conveys authentic street food atmosphere and smoky flavor, 
professional branding quality, memorable symbol, 
black background version included 
--ar 1:1 --style modern --quality 2
```

## 技术优势

### 1. **智能适配**
- 自动识别用户意图，无需手动选择风格
- 支持中英文关键词混合输入
- 多风格、多行业同时识别

### 2. **专业指导**
- 内置设计、摄影、艺术专业知识
- 针对不同行业提供定制化建议
- 符合主流AI绘图工具（Midjourney/DALL-E/SD）规范

### 3. **动态优化**
- 每次输入都会生成定制化的系统提示词
- 结合模板预设 + 动态分析 = 最优结果
- 灵活应对各种设计需求

### 4. **提升效果**
- 生成的提示词更加专业和详细
- 符合AI绘图模型的理解习惯
- 大幅提高出图质量和准确度

## 配置要求

### 必须配置AI API
要使用此功能，必须在"设置"中配置：

**云端API模式：**
- API Base URL（如 https://api.openai.com/v1）
- API Key
- 模型名称（如 gpt-4、gpt-3.5-turbo）

**或本地Ollama模式：**
- 安装并运行 Ollama（http://localhost:11434）
- 选择已下载的模型（如 llama2、mistral）

### 未配置API时
- 系统会显示警告提示
- 只能生成基础的结构化提示词
- 无法获得AI优化的专业图像提示词

## 工作流程

1. **选择设计模板**（如Logo设计专家）
2. **填写变量**（品牌名称、风格等）
3. **点击"生成提示词"**
4. **系统自动**：
   - ✅ 分析所有输入内容
   - ✅ 识别风格/行业/元素关键词
   - ✅ 构建定制化系统提示词
   - ✅ 调用AI API生成最终英文提示词
5. **获得结果**：
   - 📝 最终图像提示词（可直接用于Midjourney等）
   - 📋 一键复制
   - 🔄 可重新生成

## 注意事项

1. **输入越详细，效果越好**
   - 尽量提供风格、色彩、氛围等关键词
   - 中英文均可，系统会智能识别

2. **检查AI配置**
   - 确保API配置正确
   - 本地Ollama需先启动服务
   - 模型选择影响输出质量（推荐GPT-4）

3. **提示词语言**
   - AI最终输出为**英文**（AI绘图模型对英文支持更好）
   - 可以看到中文的结构化提示词作为参考

4. **迭代优化**
   - 如果结果不满意，可以调整输入后重新生成
   - 尝试添加更多关键词
